La présentation du territoire comprend deux pages, d'où les boutons recto verso.
Vous pouvez choisir entre l'un de ces 3 modèles de mise en page, en fonction de votre contenu (A, B ou C). Attention, ce choix est définitif, vous ne pourrez plus changer de modèle après avoir commencer la mise en page. Lisez la suite pour savoir comment choisir le modèle plus adapté. N'hésitez pas à copier votre livret avant de procéder à votre sélection.
>> Passez votre souris sur les aperçus pour savoir de quel modèle il s'agit : A, B ou C.

Le modèle A :
Ce modèle est adapté pour intégrer :
- 2 cartes ou photos de présentation de votre territoire (une carte globale + une carte sur une zone d'activité par exemple),
Ce sont des zones images qui peuvent être utilisées pour tout type de visuels de présentation du territoire.
- un témoignage d'un acteur avec une photo de type photo d'identité,
- une présentation du patrimoine architectural avec 3 photos,
- une présentation du patrimoine environnemental avec 1 photo,
- une partie sur les activités économiques avec 1 photo.

Le PLUS du modèle A :
- les 2 cartes (ou visuels) pour le recto.
- la possibilité de mettre beaucoup d'images

Le modèle B :
Ce modèle a été pensé pour intégrer :
- 1 carte de présentation de votre territoire ou 1 grande photo pour présenter le territoire.
- un témoignage d'un acteur (sans photo),
- une présentation du patrimoine architectural avec 2 photos,
- une présentation du patrimoine environnemental sans image,
- une partie sur les activités économiques sans image.

Les PLUS du modèle B :
- le grand visuel (carte ou autre) pour le recto.
- peu d'images nécessaires

Le modèle C :
Ce modèle a été conçu pour intégrer :
- une présentation de votre territoire avec 3 petites photos
- un paragraphe sans photo pour les forces et les faiblesses
- un témoignage d'un acteur avec 1 photo de type photo d'identité,
- une présentation du patrimoine architectural avec 1 petite photo,
- une présentation du patrimoine environnemental avec 1 petite photo,
- une présentation des activités économiques sans avec 1 petite photo.

Les PLUS du modèle C :
- 3 petites photos pour présenter le territoire
- la possibilité de mettre en avant les atouts et les faiblesses du territoire
